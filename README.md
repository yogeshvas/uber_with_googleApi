## Run Locally
Clone the project

```bash
  git clone https://github.com/yogeshvas/uber_with_googleApi.git
```

Go to the project directory

```bash
  cd kalesh
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
